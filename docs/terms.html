2. Service Description

Nova Core is an independent Discord RPG bot developed by SharifPoetra. Built with Node.js 22+, TypeScript, Sapphire v5, and MongoDB.

Core features:

  • RPG Core: EXP, levels, 3 classes (Warrior/Mage/Rogue) with unique passives
  • Battle Engine v2: turn-based combat, skills, cooldowns, buffs/debuffs
  • Dungeon: 100 floors with boss encounters every 10 floors
  • Activities: /hunt, /explore, /fish with drop tables
  • Crafting: /cook, /craft with recipes
  • Economy: /daily, /shop, /sell, stamina system
  • Commands: /start, /profile, /inventory, /droprate, /ping, /help
