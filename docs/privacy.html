Privacy Policy

Nova Core ("Nova") is a Discord RPG bot developed by Mohd Syarif Nasir (@SharifPoetra). Nova is built with Sapphire Framework v5 and MongoDB/Mongoose, focusing on a scalable turn-based RPG experience.

1. Data We Collect

  • Discord identifiers: User ID, username, server ID
  • RPG profile: class (Warrior/Mage/Rogue), level, EXP, HP, stamina
  • Game progress: dungeon floor (1–100, boss every 10 floors), hunt/explore/fish results, inventory, crafting materials
  • Economy: coins, daily streak, shop transactions
  • Technical logs: command usage (/start, /profile, /hunt, /explore, /fish, /cook, /craft, /dungeon, /daily, /shop, /sell), timestamps, error traces — we do not store message content

2. Purpose

Data is used to save character progression, run Battle Engine v2 (turn-based skills, cooldowns, buffs/debuffs), manage class passives, and maintain game balance.

3. Storage & Security

Data is stored in MongoDB. Access is restricted to the developer. We implement standard security practices for Node.js/TypeScript applications.

4. Data Sharing

We do not sell or share personal data. Processing is limited to Discord API (to operate the bot) and our database provider.

5. Your Rights — Deletion

Nova does not have an automated /delete-data command. To request deletion of your RPG data, contact the developer directly:

→ Discord: mohd.syarif via support server discord.gg/ZEFrfj5
Provide your Discord User ID. Requests are processed within 7 days and confirmed via Discord DM.

6. Retention

Data is retained while you use Nova. Inactive profiles (>12 months) may be archived.

7. Children's Privacy

Nova complies with Discord's 13+ age requirement.
